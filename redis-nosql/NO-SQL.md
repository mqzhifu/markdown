
|     |                |                              |                   |     |
| :-- | :------------- | :--------------------------- | :---------------- | --- |
| KV型 | REDIS/MEMCACHE | QPS非常大，性能非常高                 | 不好持久化，只可以根据 KEY查询 |     |
| 搜索型 | ES             | 写多读少，需要全文搜索                  | 吃硬件               |     |

|     |                |                              |                   |     |
