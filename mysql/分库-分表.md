# 初衷

当某个表中的数据非常大的时候，任何操作(CURL)的性能都会下降
慢的原因：
- 数据太大，扫描一次，IO太大
- 索引过大。数据量大索引也就大，UPDATE DELETE 重做索引时，会很慢
- 索引也大，并发会降低
