# 线性回归核心概念

方程：
>y = mx + b

在这个汽车重量与油耗的例子中：

y (因变量) = MPG（每加仑英里数）← 我们要预测的目标
x (自变量) = 汽车重量（磅）← 输入特征
m (斜率) = 重量每增加1磅，MPG 的变化量（应为负值）
b (截距) = 当重量为0时的理论 MPG 值

给大量数据做训练：特征值 -> 结果值
核心：计算出 b 和 m 。
>b 和 m 的值，得到一个：拟合线



损失计算： b 和 m，最佳结果与实际结果的差值。
肯定是越小越好，但要考虑可能有离群点。
梯度下降法

```python
# ==============================
# 纯Python机器学习、不使用sklearn
# 任务：根据车重量预测续航公里数
# ==============================

# 1. 训练数据
X = [1.0, 1.5, 2.0, 2.5]   # 车重量（吨）
y = [500, 400, 300, 200]   # 对应公里数

# 2. 模型参数（初始随机值）
w = 0.1    # 权重
b = 0.1    # 偏置
lr = 0.1   # 学习率

# 3. 开始训练（迭代1000次）
for epoch in range(1000):
    y_pred = []
    
    # 第一步：预测所有数据
    for x in X:
        pred = w * x + b
        y_pred.append(pred)
    
    # 第二步：计算误差总和
    dw_sum = 0
    db_sum = 0
    n = len(X)
    
    for i in range(n):
        error = y_pred[i] - y[i]
        dw_sum += error * X[i]
        db_sum += error
    
    # 第三步：更新 w 和 b（核心学习步骤）
    dw = dw_sum / n
    db = db_sum / n
    
    w = w - lr * dw
    b = b - lr * db

# 4. 输出训练结果
print("==== 训练完成 ====")
print(f"公式：y = {w:.2f} * x + {b:.2f}")

# 5. 预测新数据
weight = 1.8
result = w * weight + b
print(f"\n重量 {weight} 吨的车，预测续航：{result:.2f} 公里")

```


| 步骤 | 关键动作 | 输入 | 输出 | 核心公式/概念 | 你的控制权 |
|:----:|----------|------|------|---------------|------------|
| ① | 准备数据 | 原始记录<br>（如：车重、油耗） | X (特征)<br>y (标签) | `X = [[1.2], [1.5]...]`<br>`y = [7.2, 8.1...]` | ✅ 选特征<br>✅ 洗数据 |
| ② | 定义模型 | 特征 X | 预测函数<br>ŷ = M·x + B | `h(x) = M*x + B` | ✅ 选模型结构 |
| ③ | 前向传播 | X, M, B | 预测结果 ŷ | `ŷᵢ = M*xᵢ + B` | ⚙️ 初始参数 |
| ④ | 计算损失 | ŷ, y | 损失值 Loss | `MSE = ½n·Σ(ŷᵢ-yᵢ)²` | ✅ 选损失函数 |
| ⑤ | 反向传播 | Loss, X, ŷ | 梯度<br>∂L/∂M, ∂L/∂B | `∂L/∂M = ⅟n·Σ(ŷᵢ-y)·xᵢ` | ⚙️ 自动求导 |
| ⑥ | 参数更新 | M, B, 梯度, lr | 新参数<br>M', B' | `M' = M - lr·∂L/∂M`<br>`B' = B - lr·∂L/B` | ✅ 调学习率<br>✅ 迭代次数 |
| ⑦ | 循环迭代 | 新 M, B | 收敛的参数 | `while Loss > threshold:` | ✅ 设早停条件 |
| ⑧ | 输出模型 | 最终 M, B | 可用模型<br>y = M·x + B | `return {"M": M, "B": B}` | ✅ 本地部署 |
