

普通格挡：掉血（虚血），耗精

完美格挡：不掉血、只耗精、给敌人削韧

敌人红光/重击：只能完美格挡，不能闪、不能硬挡。

虚血机制：非完美格挡会掉血（虚血），即时反击能回血。
>如果没能即时反击，又被打了，虚血立刻会被扣掉
>如果没能即时反击，过了一段时间，虚血立刻会被扣掉

韧性/处决：打空敌人韧性，敌人的血条外层显示白条→重攻击破防→上前处决
>没有主动打破防再处决，得出现可能破防后，再蓄力重击，才是真正破防，可处决

脉冲电池/血瓶：用完最后 1 瓶变灰，攻击 / 处决敌人可充能

# 属性

生命力：血条
持久力：精力条
负重：走路速度、翻滚速度
动力：力气 / 力量
技术： 敏捷 / 技巧
进步：魔力 / 才智


虚构栏：蓝条/技能条
防御回复




|负重档位|负重占比|移动速度|翻滚距离|翻滚后摇硬直|
|:--|:--|:--|:--|:--|
|轻负载|＜60%|最快|最远|最短|
|中负载|60%～80%|中等|中等|中等|
|重负载|＞80%|最慢|最短|最长|

当前负重 = 右手武器+左手武器+护身符(2个)+防御零件(骨架、转换器、弹药筒、内衬)

 
骨架：物理减伤
转换器：属性减伤 + 异常抗性（火 / 雷 / 酸 / 腐蚀）
弹药筒 / 循环器：特殊异常抗性（眩晕、失衡、断裂、干扰）
内衬：单一物理减伤（和骨架叠加）最轻，一般选高减伤、低负重的

扰乱：满了直接死
# 武器


普通武器 + 特殊武器（打BOSS后，兑换的）

武器：剑刃+手柄 组成。
>所有武器都是由2部分组成的，可拆分开，也可任意再组合成新武器

刃：基础攻击力、处决模组、刃战技、格挡减伤、充能速度。
>可以用石头来升级

手柄：动作模组（轻 / 重 / 蓄力 / 闪避攻）、属性补正（动力 / 技术 / 进步）、柄战技、负重、攻击距离
可以使用道具，改变  武器修正

活人偶斧子+荆棘手柄（菜刀头）
火蜥蜴匕首+酸性水晶矛柄
暴君杀人魔匕首+城市长矛柄
线圈的柄+转轮







动力手柄≠直接加攻击力，它提升的是「力量属性」对伤害的加成系数

最终攻击力 = 武器基础面板 + (力量属性 × 动力加成系数)

# 缺点

| 缺点条目       | 详细说明                          |
| ---------- | ----------------------------- |
| 闪避机制弱势     | 无传统翻滚，闪避距离短、无敌帧极少；仅能靠后期技能小幅弥补 |
| 普通防御代价大    | 防御会扣除虚血、同时消耗大量精力              |
| 武器韧性偏低     | 武器易崩韧，需要频繁修理；后期技能可小幅改善        |
| 战技绑定固定     | 武器战技无法拆卸更换                    |
| 无绝对弹反      | 没有通用完美弹反机制，仅部分武器自带化解技能        |
| 掉血机制苛刻     | 格挡才会掉虚血，直接挨打全额掉血              |
| BOSS 血量偏高  | 多数 BOSS 拥有双管血量                |
| 处决机制别扭     | 无明显真正失衡阈值，临近失衡还需补蓄力重击才能触发处决   |
| BOSS 快慢刀泛滥 | 大量 BOSS 高频快慢刀，难预判节奏           |
# 优点

| 优点条目    | 详细说明                             |
| ------- | -------------------------------- |
| UI 美术风格 | 维多利亚蒸汽朋克复古，质感出众                  |
| 剧情叙事    | 碎片化叙事但信息完整，剧情逻辑易懂、不晦涩            |
| 地图设计    | 地图结构简单、路线引导清晰不绕路，捷径布局合理          |
| 唱片系统    | 唱片搭配画面配乐，氛围感渲染极强                 |
| 形象彩蛋    | 游戏内小猫形象可爱，细节加分                   |
| NPC 塑造  | NPC 人设鲜活、有完整故事线和情绪变化，人物不工具人，代入感强 |
