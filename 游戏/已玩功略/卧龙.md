# 畏缩

一种异常状态：类似于"胆怯"或"胆怯僵直"

当敌人累积了足够的畏缩值后，畏缩状态触发
触发后，敌人在受到攻击时会出现硬直/僵直（平时不会畏缩的敌人也会畏缩）
畏缩和属性异常状态（中毒、灼伤等）机制相同，累积到一定程度后发动

哪些武器能累积畏缩
大刀、戟、枪、矛：普通攻击时会使敌人累积畏缩
棍杖：畏缩与属性异常状态相同，累积到一定程度后发动

# 死亡惩罚

死亡后
- 掉一半的真气
- 你的士气等级归零（"不屈等级"还在）
- 并让敌人升级
- 钉死你的那个敌人身上会冒紫光

复活后：
- 被普通敌人击杀 → 击败该敌人复仇成功即可取回
- BOSS击杀 → 只要再次触发BOSS战就能取回（卷土重来），无需复仇
- 复仇途中再次死亡 → 真气彻底消失
- 摔死/环境死亡 → 没有仇家，真气直接没收

坠崖：并不会立刻死，会扣一部分血

# 气势

气势（条）：红条(左)+ 蓝色(右)
红条（负气势）：挨打、防御、使用绝技/奇术会消耗
蓝条（正气势）：攻击命中、成功化解会增加

注：轻攻击打中敌人，会减少红条累积，为0，才开始涨蓝条

气势慢慢都会归0



消耗正气势
- 进攻（消耗正气势）：重攻击、绝技、奇术
- 防御（消耗负气势）：化解、闪避、格挡

重攻击：消耗正气势，威力随消耗量提升，能贯穿敌人格挡
>没有气势也能重攻击，但不能贯穿敌人格挡，威力较小，敌人几乎不掉气势


当敌人的气势条被打到负值（负气势）下限，敌人会进入"气势受挫"状态（浑身冒橙黄光），此时按下气势攻击键即可触发绝脉，造成巨额伤害并降低敌人士气等级。
打BOSS基本就是靠反复化解→削减气势→绝脉来输出的。


|操作|消耗量|说明|
|:--|:--|:--|
|闪避/翻滚|中等|每次翻滚都会消耗|
|化解|少量|单次化解消耗不多，但连续化解会越耗越多|
|绝技（武技）|较大|武器专属技能，消耗明显|
|奇术（法术）|较大|五行法术，不同奇术消耗不同|
|气势攻击（重攻击）|中等|消耗正气势，正气势不够时会转为消耗负气势|
|被敌人攻击命中|大|挨打是最主要的气势消耗来源|
|格挡（按住防御）|持续消耗|持续按住防御会不断消耗气势|


| 方法                    | 效果                      |
| :-------------------- | :---------------------- |
| 成功化解                  | 大幅增加正气势（蓝色），正气势会优先抵消负气势 |
| 攻击命中敌人                | 增加正气势，间接抵消负气势           |
| 化解转击（R1 + ○ / RB + B） | 快速替换武器反击，能一口气清零负气势，非常实用 |
| 绝脉（处决）                | 成功处决后负气势直接清零            |
| 什么都不做                 | 负气势会缓慢自动恢复，但非常慢，战斗中不现实  |


# 士气级别


每个关卡独立的"等级压制"系统   
>也可以理解为：攻击/被打 的伤害位数

血条上面有个数字，显示自己的士气级别。
敌人头上也有数字，显示敌人的士气级别：
- 绿色：等于或低于
- 黄色：高于1~4级
- 红色：高于玩家5级以上（2倍及以上的伤害）


士气等级差：决定你打别人的伤害和别人打你的伤害
>等级差3级以上：伤害倍率非常明显

进入关卡时，你的士气等级从 0 开始，最高 25级

增加士气
- 击杀敌人（普通+精英），精英怪和武将击杀后士气提升更多
- 敌人头上有数字，比你士气高的敌人击杀后提升更多
- 插旗
- 复仇

消减士气：
- 死亡
- 被敌人红光技能打中


# 负重 

0-30%: 敏捷A 翻滚不消耗气势
30-70%:敏捷B 翻滚消耗气势 1.25倍
70-100%:敏捷C 翻滚动作变慢 1.75倍
100%以：敏捷D 无法翻滚 3.75倍

负重越高：攻击/防御 所有操作，消耗均会更高
