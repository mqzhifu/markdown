# 供养

把没用的魂核 / 装备拆掉换强化材料 / 升级经验，是仁王 3 里必用的养成循环

## 魂核供养

神社 → 魂代 → 供养：把多余、重复、低级的魂核喂掉
得到：常世的残思（核心强化素材）
残思用来：
- 魂核觉醒 / 升级（提高伤害、解锁被动）
- 守护灵觉醒 / 升级（加攻加防、开守护灵大招）
- 偶尔还能出妖怪素材，用于锻造 / 改造装备

## 装备供养

神社 → 整顿 → 奉献武具：把垃圾装备（白 / 绿、没用属性）奉献，得到：精华

供养 vs 卖 vs 分解:
供养（奉献）：装备→精华(升级属性)；魂核→残思（升级魂核）
卖出：装备→钱（买道具）
分解：装备→锻造素材（做装备）

# 武士


核心玩法：冬月 啄技 残心 架势流转 精准格挡

残心
- 回收精力。按RB  恢复 虚精力（红色变白的条） 条
- 同时可拔除
- 取消攻击后摇（跟回收精力有点冲突）

啄技：攻击和防御可积累啄技，满后：重攻击或武技即可触发啄技，期间：
- 武器会带白光
- 强攻击和武技伤害加大（跟|技|挂钩，最大加3%）
- 耗精减少
- 重攻击或武技增加伤害（发动期间不耗精）

流转
- 切换架势（上、中、下）段,RB+Y RB+X RB+A
- 额外回复精力
- 取消攻击后摇

精准格挡：不消耗精力、打断敌人普攻节奏

# 敌人-失衡-处决


## 人类

打空精力条，就应直，可直接处决。
敌人不被攻击，就可以自然快速恢复
所以，打人型怪，最好的方式就是一直攻击，不能停，很快打出处决

## 妖怪

没有精力条，有妖力条（紫色条）

- 正常状态下：不能自然恢妖力条（不用像人型怪，需要一直连续打）
- 在常世圈里，或常暗环中 才能自然恢复妖力条
- 虽然不能自然恢复，但消耗的也慢，个别招式才耗精

打空妖力条不会立刻应直，会进入脆弱状态
- 妖力条会变成粉红色
- 妖怪还是能正常攻击，但没那么频繁
- 会失去霸体，正常攻击+武技，能打出小失衡，赶紧上去输出
- 再打空粉红色的条，才会出现应直，可处决

# 异常

同时两种异常：进入混沌状态（身上紫光），受到的伤害大幅增加，精力恢复也慢
两种：受伤效果40%
三种：受伤效果70%
BOSS进入常暗状态：画面变灰白，攻击欲望增加，消除异常状态

识破：可以回精


## 异常状态

雷 火 水 风 ：可以让敌人上异常状态
火：持续掉血
水：伤害增加
雷：降低攻速、移速。
风：精力消耗变高、承受精力伤害增加。


# 忍者

## 优势

- 耗精低 
- 背刺伤害高（2.6倍）

>可以配合：被动技能(常世驱散)，破掉常世的圈

正面伤害较低，得绕背输出
