# 处决

敌人身上出现红点，正面处决可回满精力，背面处决不回精但伤害高（比正面多50%），
>正常背击：也加伤害，背后蓄力重攻击

红点：慢走背后/精力条空了，可处决
虚弱：精力条空了，可处决


# 韧性/失衡/反制/反弹/防御/闪避

打失衡后：可连续攻击

失衡条件：
- 小怪韧性低，连续打即可
- 反制
- 反弹

反制：L1+B,红光可以反制。反制成功：回满精力，有大伤害
>只能红光时才起作用

反弹：L1+Y
>红光不能反弹


即时防御/完美防御：不耗精，在敌人即将攻击的时候按防御键。红光不可防御
即时闪避/完美闪避：耗精，强化下一次普攻击

# 幽魂能力

解锁8 个幽魂。每个幽魂有各种的技能(被动或主动)。
幽魂升级：提高其技能效果。
- 日常打怪即可升级
- 最高 20 级
- 收集鬼气材料觉醒，觉醒后解锁更强二级被动


幽魂同行：被动 (最多装备 1 个同行幽魂)
解锁：击败斗鬼 BOSS、拿到幽魂遗物，交给布雷德解锁

助力之魂：打BOSS的时候，可召唤



# 技能点

升级技能点：打怪、武器熟练度、完美防御，会有一定阀值，不能无限升

斗志：血条上方黄色，默认给6个。
释放某些技能会消耗斗志，但不耗精

标枪：非战斗状态(红点)/弱点状态(打头)，可增加伤害

# 武器

刀斧：攻速快、伤害低、削韧低、较灵活
巨剑：攻速慢、伤害高，削韧高、笨重
枪



# 属性点

活力：血+抗性，少许负重
耐力：精力条
力量：巨剑，武器补正
意志：刀斧，武器补正
潜力：枪，武器补正

这是不管什么武器：都得加武器，因为加负重

负重：尽量保持A，差一点B，一但B以下，敏捷太低

# 装备

可装备：
- 武器
- 头+胸+手+腿+鞋
- 项链+戒指

装备属性有2类：随机特性/词条 + 装备特性
装备特性：不可更改

随机特性/词条：耗费魂可洗词条
洗词条好坏选择：
- 优先：攻击力、伤害增加、攻击倍率，
- 次之：精力回复速度、最大精力，
- 最后：防御力、生命力
洗词条价格：第一次最便宜，越往后越贵，可存档洗。也可以重置词条


| 颜色  | 中文名称 | 随机特性/词条     |
| --- | ---- | ----------- |
| 白色  | 普通   | 无额外加成       |
| 蓝色  | 优秀   | 1条随机        |
| 紫色  | 稀有   | 2条随机        |
| 粉色  | 独特   | 3条随机        |
| 橙色  | 史诗   | 3条随机 + 1条固定 |


套装
- 每多穿一件就会激活：装备特性中的一个词条
- 最多是6件，装备5件+1件武器
- 可混穿：两个套装，属性是叠加的



品级强化材料（黄金魔方）：

| 材料名称   | 强化路径          | 效果说明            |
| ------ | ------------- | --------------- |
| 微光黄金魔方 | 稀有(紫) → 独特(粉) | 解锁第3条随机词条       |
| 闪耀黄金魔方 | 独特(粉) → 史诗(橙) | 解锁第4条属性（+1固定词条） |
| 灿烂黄金魔方 | 史诗(橙) → 传说(棕) | 解锁传说专属加成（新周目）   |
补正转换材料（光辉魔石）

| 魔石等级 | 可获得的补正组合示例                           | 适用场景       |
| ---- | ------------------------------------ | ---------- |
| 下级   | `[意志B/力气C]`、`[力量B/意志C]` 等 B~C 品级组合   | 前期过渡，低成本试错 |
| 中级   | `[意志B+/力气C]`、`[力量B/意志B]` 等 B+~C 品级组合 | 中期优化，平衡输出  |
| 上级   | `[意志A/力气C]`、`[力量B+/意志B]` 等 A~C 品级组合  | 毕业追求，极限搭配  |
