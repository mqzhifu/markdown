# 人物抵抗力

|     |           |          |     |
| :-- | :-------- | :------- | --- |
| 免疫力 | 毒性、腐败(猩红) | 腐败骑士、女武神 |     |
| 健壮度 | 出血、冻伤     | 冰系魔法     |     |
| 理智度 | 催眠、发狂     | 癫狂       |     |
| 抗死度 | 死亡(骤死)    | 咒蛙       |     |

数值越高，越不容易陷入中毒、腐败状态。
抵抗力计算：人物属性值+防具的抵抗力=效果总和
