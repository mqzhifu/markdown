# 供养

把没用的魂核 / 装备拆掉换强化材料 / 升级经验，是仁王 3 里必用的养成循环

## 魂核供养

神社 → 魂代 → 供养：把多余、重复、低级的魂核喂掉
得到：常世的残思（核心强化素材）
残思用来：
- 魂核觉醒 / 升级（提高伤害、解锁被动）
- 守护灵觉醒 / 升级（加攻加防、开守护灵大招）
- 偶尔还能出妖怪素材，用于锻造 / 改造装备

## 装备供养

神社 → 整顿 → 奉献武具：把垃圾装备（白 / 绿、没用属性）奉献，得到：精华

供养 vs 卖 vs 分解（简单分清）
供养（奉献）：装备→精华（升级）；魂核→残思（强化）
卖出：装备→钱（买道具）
分解：装备→锻造素材（做装备）

# 敌人-失衡-处决


## 人类

打空精力条，就应直，可处决

### 妖怪

没有精力条，有妖力条。
攻击闪避是不耗精的，个别招式耗精
在常世/常暗的圈里，恢复精力
打空妖力条不会立刻应直，但会失去载体状态，继续任意攻击才会应直，再继续攻击，会缩小妖气的上限，再把这个打空，才可以处决
