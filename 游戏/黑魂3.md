# 概述

version:  游戏版本 1.15.2 / 规则 1.35

所有衣服防御是一样的，只有减伤不同。

血：27和39是两个阀值
负重：不超70%
双持：力量1.5倍

刷材料：
火炎壶到鸟巢交换楔大碎片，黑火炎壶还给楔石块

https://www.bilibili.com/read/cv4581925/

黑骑士盾，最强中盾（力量需要18）

元素碎片：增加元素瓶使用的次数
>铁匠处增加

燃烧不死人遗骨：增加元素瓶的恢复量
>只能在传火祭祀场处的营火旁进行
