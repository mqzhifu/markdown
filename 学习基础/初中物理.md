# 声


物体的振动：发出声音
声音传播途径：空气、气体、固体、液体
>太空中没有空气 ，真空状态，只能用无线电。桌子也是能传音的。水中也能听见岸上的声音

声波：物体撼动，带动周边空气振动，像水波一样。
声速：声音传播的速度
>比如：闪电会先被眼睛捕捉到，但是声音要延迟一会，被耳朵捕获

声速的传播：取决于发生体振动大小有关，还有传播的介质。

>室内说话就比室外声音更大，因为室内会有回声。电影院通过回声机制制造更好的音效。
>空气传播的速度是很慢的，水 冰 铜 大理石 铝 铁 都更快


耳朵是人类接收声波并解析成被神精指令，传输给大脑的‘部件’

频率 frequency :物体每秒振动的频率  hertz
程度：物体振动的幅度。振幅 amplitude

各种动物的听觉各不相同，有的对低频敏捷，有的对高频敏感
>人也是动物

次声波：低于20HZ，地震、火山喷发、台风、海啸
>大象使用次声波交流，人类是听不见的。有灾难的时候，动物也会提前感知到，人类却不知道

# 运动和力

牛顿第一定律：任何物体在没受到阻力的作用时，保持：静止状态或匀速直线运动状态
惯性：由第一定律推导得出。
>小球与地面中间有一层铁片，抽出铁片，小球是向下坠落。
>汽车中，突然急刹车，人是向前运动。

二力平衡：物体受力了，但并不一定发生运动，可能是静止的。产生了两者的平衡
>吊灯是静止的，但灯有向下的重力，顶承受了这个重力，二力平衡
>在同一物体上，两个方向，施加相同的力，即：二力平衡

摩擦力：两个物体相互接触
摩擦力的关键：
- 受力面积。物体接触摩擦的面积大小
- 压力。施加摩擦力的大小压强

压强 pressure：物体所受的压力大小与受力面积之比
p(pascal) =  F / S = 压力(牛) / 受力面积(米) = 物体对物体的产生的压强

加大压强：增加受力面积
>汽车多加轮子，可以拉更多的货

减少压强：减少受力面积
>使用有轮子运输，减少使用的力

液体装在容器中，会流动，对容器所有角度都有压强。
同一深度，压强一样。越深，压强越大

s( 物体平面面积)  h(高度)
p = pgh

流体：具有流动性的物体，如：空气+液体

# 浮力

一块铁，在水中的重力小于在非水中的重力；
水中有压强，且越深压强越大。
在水中：铁的上面的压强小于下面的压强，于是产生了浮力

浮力大小关系：液体是什么类型、浸入液体的物体的体积

物体扔进液体里，排出来的水，也可以间接证明物体的质量是多少

阿基米德原理 

浮力计算：
- 物体在非液体中的重力
- 物体在液体中的策略
- 物体排出的液体的重力（F=G）

物体在水中的状态？
- 浮在上面
- 浸在中间
- 沉到底部

已知：物体在水中，受到策略向下，受到浮力向上两种力
 - 浮力 大于 重力 ，浮起
 - 浮力 等于 重力，中间
 - 浮力 小于 重力，沉底

除了基本的重力浮力外，还跟液体的密度有关系
- 物体密度小于液体密度，上浮
- 物体的密大于液体密度，沉底
